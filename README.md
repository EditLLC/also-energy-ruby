# AlsoEnergy

This is a Ruby wrapper for the Also Energy Production Data SOAP API Version 1.6.1

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'also_energy'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install also_energy
